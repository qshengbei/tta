# 消息模板设计

## 1. 模板类型

| 类型 | 描述 | 场景示例 |
|------|------|----------|
| orderStatusChange | 订单状态变化通知 | 支付成功、发货、确认收货、订单取消等 |
| activity | 活动通知 | 新品上架、促销活动、限时折扣等 |
| system | 系统通知 | 系统更新、维护通知、重要公告等 |
| restock | 补货通知 | 商品补货、到货提醒等 |

| completed | 订单完成 | 您的订单 #{{orderNumber}} 已完成，感谢您的购买，期待您的再次光临 | 所有 |

## 6. 模板变量说明

| 变量名 | 类型 | 描述 | 适用模板 |
|--------|------|------|----------|
| orderNumber | string | 订单编号 | 订单状态变化通知 |
| amount | number | 支付金额 | 支付成功通知 |
| trackingNumber | string | 物流单号 | 发货通知 |
| deliveryType | string | 配送方式 | 发货通知 |
| reason | string | 售后原因 | 售后申请通知 |
| result | string | 售后结果 | 售后完成通知 |
| productName | string | 商品名称 | 新品上架、限时折扣、补货通知 |
| price | number | 商品价格 | 新品上架通知 |
| activityName | string | 活动名称 | 促销活动通知 |
| description | string | 活动描述 | 促销活动通知 |
| discount | number | 折扣力度 | 限时折扣通知 |
| countdown | string | 倒计时 | 限时折扣通知 |
| version | string | 系统版本 | 系统更新通知 |
| features | string | 新功能描述 | 系统更新通知 |
| time | string | 维护时间 | 系统维护通知 |
| duration | string | 维护时长 | 系统维护通知 |
| title | string | 公告标题 | 重要公告通知 |
| content | string | 公告内容 | 重要公告通知 |
| stock | number | 库存数量 | 补货通知 |

## 8. 模板使用说明

1. **模板选择**：根据通知类型和场景选择对应的模板
2. **变量替换**：将模板中的变量占位符替换为实际值
3. **通知发送**：通过 sendNotification 云函数发送通知
4. **微信模板消息**：如需发送微信模板消息，需配置对应的微信模板ID

## 9. 示例

### 订单支付成功通知
- 模板：orderStatusChange + paid
- 变量：{"orderNumber": "202604090001", "amount": "99.99"}
- 标题：支付成功
- 内容：您的订单 #202604090001 支付成功，支付金额 99.99 元，我们正在处理

### 新品上架通知
- 模板：activity + newProduct
- 变量：{"productName": "春季新款连衣裙", "price": "199.99"}
- 标题：新品上架
- 内容：新品 春季新款连衣裙 已上架，限时优惠价 199.99 元，点击查看详情
