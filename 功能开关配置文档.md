## 商品列表页面功能开关

### 页面路径
`pages/category/index.js`

### 1. enableScrollHideTopBar - 滑动隐藏顶部栏开关

#### 功能描述
控制商品列表页面的顶部栏（标签栏和/或排序栏）是否随滚动隐藏/显示。

#### 配置位置
```javascript
data: {
  enableScrollHideTopBar: false, // 功能开关
  scrollHideMode: 'tabs',        // 隐藏模式
  isTopBarVisible: true,         // 顶部栏当前可见状态
}
```

#### 取值说明

**enableScrollHideTopBar**：

| 值 | 效果 |
|-----|------|
| `true` | 开启滑动隐藏功能 |
| `false` | 关闭滑动隐藏功能（默认） |

**scrollHideMode（仅在 enableScrollHideTopBar=true 时生效）**：

| 值 | 效果 |
|-----|------|
| `'both'` | 隐藏标签栏和排序栏 |
| `'tabs'` | 只隐藏标签栏（保留排序栏）（默认） |
| `'sort'` | 只隐藏排序栏（保留标签栏） |

#### 影响范围

**WXML 布局影响**：
- 标签栏位置（`top` 属性）
- 排序栏位置（`top` 属性）
- 容器内边距（`padding-top` 属性）
- 滚动区域高度（`scroll-view` 的 `height` 属性）

**JS 逻辑影响**：
- `onProductsScroll()` 滚动监听事件处理逻辑

#### 使用场景

| 场景 | 推荐配置 | 原因 |
|-----|---------|------|
| 追求滑动流畅度 | `enableScrollHideTopBar: false` | 避免频繁重排导致的卡顿 |
| 追求屏幕空间利用率 | `enableScrollHideTopBar: true, scrollHideMode: 'both'` | 上滑时隐藏所有顶部栏，显示更多商品 |
| 需随时切换排序 | `enableScrollHideTopBar: true, scrollHideMode: 'tabs'` | 保留排序栏，隐藏标签栏 |
| 需随时切换标签 | `enableScrollHideTopBar: true, scrollHideMode: 'sort'` | 保留标签栏，隐藏排序栏 |
| 测试/调试 | `enableScrollHideTopBar: false` | 顶部栏始终可见，便于测试 |

#### 历史变更

| 版本 | 变更内容 |
|-----|---------|
| v1.0 | 初始实现，默认开启滑动隐藏整个顶部栏 |
| v1.1 | 添加开关控制，默认关闭滑动隐藏 |
| v1.2 | 添加 scrollHideMode 配置，支持三种隐藏模式 |

## 维护规范

### 添加新开关

1. 在对应页面的 `data` 对象中添加开关变量
2. 在 `onLoad` 或初始化位置设置默认值
3. 在逻辑代码中添加开关判断
4. 在本文档中添加开关说明

### 修改开关默认值

1. 修改代码中的默认值
2. 更新文档中的默认值说明
3. 在历史变更中记录修改

### 废弃开关

1. 在代码中标记为废弃（添加 `// @deprecated` 注释）
2. 在文档中添加废弃标记
3. 后续版本中移除相关代码
