# 通用时间显示规则

## 规则
- **当天**：显示"时分"格式（如：14:30）
- **昨天**：显示"昨天 时分"（如：昨天 05:09）
- **前天**：显示"前天 时分"（如：前天 05:09）
- **更久的时间**：
  - 如果是当年，显示"月-日 时分"（如：04-15 05:09）
  - 如果是往年，显示"年-月-日 时分"（如：2025-04-15 05:09）

## 应用场景
- 消息页面的通知卡片
- 通知详情页
- 通知列表页
- 客服消息卡片

## 实现方式
使用 `formatTimeByRule` 函数实现，该函数会根据时间差自动选择合适的显示格式。
