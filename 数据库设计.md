## 2. 集合结构

### 2.1 products (商品集合)

| 字段名 | 数据类型 | 描述 | 必需 |
|-------|---------|------|------|
| _id | string | 商品ID（自动生成） | 是 |
| name | string | 商品名称 | 是 |
| price | number | 商品价格 | 是 |
| stock | number | 商品库存 | 是 |
| typeId | string | 类别ID（关联product_types集合） | 是 |
| categoryId | string | 系列ID（关联category集合） | 否 |
| materialId | string | 布料ID（关联material集合） | 否 |
| coverImage | string | 商品主图（云存储文件ID） | 是 |
| images | array | 商品图片数组（云存储文件ID） | 否 |
| description | string | 商品描述 | 否 |
| status | string | 商品状态（on: 上架, off: 下架） | 是 |
| tags | array | 商品标签数组 | 否 |
| sortOrder | number | 排序顺序 | 否 |
| createdAt | date | 创建时间 | 是 |
| updatedAt | date | 更新时间 | 是 |
| isDeleted | boolean | 是否删除 | 是 |
| isNew | boolean | 是否新品 | 否 |

## 3. 关联关系

| 关系类型 | 关联集合 | 关联字段 | 描述 |
|---------|---------|---------|------|
| 商品-类型 | products ← product_types | typeId ← _id | 商品属于某个类型 |
| 商品-系列 | products ← category | categoryId ← _id | 商品属于某个系列 |
| 商品-布料 | products ← material | materialId ← _id | 商品使用某个布料 |
| 类型-父类型 | product_types ← product_types | parentId ← _id | 二级类型属于一级类型 |
| 订单-用户 | orders ← users | userId ← _id | 订单属于某个用户 |
| 购物车-商品 | cart ← products | productId ← _id | 购物车项关联商品 |
| 购物车-用户 | cart ← users | _openid ← openid | 购物车项属于某个用户 |
| 地址-用户 | adressInfo ← users | userId ← _id | 地址属于某个用户 |
| 消息-用户 | messages ← users | userId ← _id | 消息属于某个用户 |
| 会话-用户 | sessions ← users | userId ← _id | 会话属于某个用户 |
| 补货-商品 | replenishment ← products | productId ← _id | 补货关联商品 |

## 5. 数据安全

| 集合名称 | 读写权限 | 描述 |
|---------|---------|------|
| products | 管理员可读写，用户可读 | 商品信息对用户可见，只有管理员可修改 |
| product_types | 管理员可读写，用户可读 | 类型信息对用户可见，只有管理员可修改 |
| material | 管理员可读写，用户不可读 | 布料信息只对管理员可见 |
| category | 管理员可读写，用户可读 | 系列信息对用户可见，只有管理员可修改 |
| users | 自己可读，管理员可读写 | 用户只能查看自己的信息，管理员可管理所有用户 |
| orders | 自己可读，管理员可读写 | 用户只能查看自己的订单，管理员可管理所有订单 |
| settings | 管理员可读写，用户可读 | 设置信息对用户可见，只有管理员可修改 |
| cart | 自己可读写，管理员可读写 | 用户只能管理自己的购物车，管理员可管理所有购物车 |
| banner | 管理员可读写，用户可读 | 轮播图对用户可见，只有管理员可修改 |
| adressInfo | 自己可读写，管理员可读写 | 用户只能管理自己的地址，管理员可管理所有地址 |
| customer_service_status | 管理员可读写，用户可读 | 客服状态对用户可见，只有管理员可修改 |
| messages | 自己可读写，管理员可读写 | 用户只能查看自己的消息，管理员可管理所有消息 |
| notifications | 自己可读写，管理员可读写 | 用户只能查看自己的通知，管理员可管理所有通知 |
| replenishment | 管理员可读写，用户不可读 | 补货信息只对管理员可见 |
| sessions | 自己可读写，管理员可读写 | 用户只能查看自己的会话，管理员可管理所有会话 |
| notification_templates | 管理员可读写，用户不可读 | 通知模板只对管理员可见，用户不需要直接访问 |
