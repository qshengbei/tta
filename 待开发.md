# 待开发功能

## 1. 小程序订阅消息功能

### 功能描述
实现小程序订阅消息功能，通过微信官方渠道推送消息到用户微信，提高通知的及时性和触达率。

### 开发步骤

#### 1.1 申请订阅消息模板
- 登录微信公众平台（https://mp.weixin.qq.com/）
- 进入"功能" -> "订阅消息"
- 选择合适的模板类型，申请所需的订阅消息模板
- 记录模板ID，用于代码中调用

#### 1.2 修改云函数
- 更新 `sendNotification` 云函数中的 `sendWechatTemplateMessage` 函数
- 将其重命名为 `sendSubscribeMessage`
- 使用 `cloud.openapi.subscribeMessage.send` API 发送订阅消息
- 调整参数格式以匹配订阅消息的要求

#### 1.3 前端实现
- 在小程序中添加 `wx.requestSubscribeMessage` API 调用
- 向用户请求订阅权限
- 处理用户同意/拒绝的情况

#### 1.4 测试和优化
- 测试订阅消息的发送和接收
- 优化用户体验，确保订阅流程顺畅
- 处理各种异常情况

### 相关文件
- `cloudfunctions/sendNotification/index.js`
- 前端页面中需要请求订阅权限的组件

### 依赖条件
- 小程序服务类目变更为电商类型
- 申请到合适的订阅消息模板

### 预期效果
- 用户可以收到微信推送的实时通知
- 通知内容清晰明了，与本地通知保持一致
- 提高用户的参与度和满意度
